# 구글 스프레드시트 기반 수주 및 재고 자동화 시스템

## 포트폴리오 개요

엑셀로 관리하던 수주, 발주, 재고 데이터를 웹 입력 폼과 구글 스프레드시트 자동화 구조로 전환하는 데모입니다. 직원이 웹에서 입력하면 지정된 시트에 수주 현황, 입고 예정, 재고 차감 내역이 자동 반영되는 구조를 가정했습니다.

이 데모는 실제 고객 데이터나 운영 시스템이 아닌, 위시켓 포트폴리오 제출을 위한 유사 구조 구현 예시입니다.

## 작업 범위

- 기존 엑셀 업무 흐름 분석을 가정한 데이터 구조 설계
- 수주/발주 입력 폼 구현
- 재고 계산 및 적정 재고 표시 로직 구현
- Google Sheets API 연동을 고려한 데이터 처리 구조
- 사용자용 경량 웹 화면 구성

## 주요 기능

- 수주 정보 입력 및 수주현황 반영
- 수입 발주 정보 입력 및 진행수량 반영
- 판매 완료 시 재고 차감
- 현재 재고 + 입고 예정 - 수주 수량 기반 부족 품목 식별
- 부족 품목 별도 리스트 표시

## 기술적 주안점

기존 엑셀 업무 방식을 최대한 유지하면서 실시간 공유와 자동 계산만 개선하는 구조를 기준으로 했습니다. 사용자가 별도 교육 없이 입력할 수 있도록 단순한 폼 UI를 구성했고, 향후 ERP 연동이나 별도 DB 전환이 가능하도록 데이터 처리 단계를 분리하는 방향을 가정했습니다.

## 실서비스 확장 방향

- Google Sheets API 인증 및 지정 시트 쓰기 연동
- 품목 마스터, 업체 마스터, 단가표 시트 연동
- 동시 입력 충돌 방지
- 작업 로그 및 수정 이력 관리
- 관리자용 사용 매뉴얼 제공
